Diagramas de Flujo

1. Suma de dos números

        graph TD;
            A[Inicio] --> B[Leer num1]
            B --> C[Leer num2]
            C --> D[suma = num1 + num2]
            D --> E[Escribir suma]
            E --> F[Fin]

2. Quicksort

        graph TD;
            A[Inicio] --> B[Si inicio < fin]
            B -->|Sí| C[Particionar lista]
            C --> D[Llamar Quicksort izquierda]
            D --> E[Llamar Quicksort derecha]
            E --> F[Fin]
            B -->|No| F

3. BFS en un grafo

        graph TD;
            A[Inicio] --> B[Crear cola vacía]
            B --> C[Añadir nodo inicial]
            C --> D[Marcar nodo como visitado]
            D --> E[Mientras cola no esté vacía]
            E -->|Sí| F[Extraer nodo]
            F --> G[Para cada vecino no visitado]
            G -->|Sí| H[Marcar y agregar a cola]
            H --> E
            G -->|No| I[Fin]

4. Algoritmo de Dijkstra

        graph TD;
            A[Inicio] --> B[Inicializar distancias a infinito]
            B --> C[Cola de prioridad con nodo inicial]
            C --> D[Mientras cola no esté vacía]
            D -->|Sí| E[Extraer nodo con menor distancia]
            E --> F[Para cada vecino]
            F -->|Sí| G[Actualizar distancia si es menor]
            G --> H[Añadir a la cola]
            H --> D
            F -->|No| I[Fin]

5. Búsqueda Binaria

        graph TD;
            A[Inicio] --> B[Definir inicio y fin]
            B --> C[Mientras inicio <= fin]
            C -->|Sí| D[medio = (inicio + fin) / 2]
            D --> E[Si lista[medio] == objetivo]
            E -->|Sí| F[Retornar medio]
            E -->|No| G[Si lista[medio] < objetivo]
            G -->|Sí| H[inicio = medio + 1]
            G -->|No| I[fin = medio - 1]
            H --> C
            I --> C
            C -->|No| J[Retornar -1]
            J --> K[Fin]
