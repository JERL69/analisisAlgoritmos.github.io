Pseudocódigo Mochila

I. INICIALIZACIÓN

    Definir LISTA objetos ← []
    Definir NUMERO capacidadMochila ← 10

II. FUNCIÓN agregarObjeto()
    Solicitar al usuario ingresar nombre, valor y peso
    Leer nombre, valor, peso
    SI nombre está vacío O valor/peso no son válidos O son menores o iguales a 0 ENTONCES
        Mostrar mensaje de error
        RETORNAR
    FIN SI
    
    Definir NUMERO ratio ← valor / peso
    Agregar {nombre, valor, peso, ratio} a objetos
    Agregar nueva fila a la tabla con los datos ingresados
FIN FUNCIÓN

III. FUNCIÓN calcularMejorCombinacion()
    Definir NUMERO pesoTotal ← 0
    Definir NUMERO valorTotal ← 0
    Definir LISTA seleccionados ← []
    
    Ordenar objetos por ratio en orden descendente
    
    PARA CADA obj EN objetos HACER
        SI pesoTotal + obj.peso ≤ capacidadMochila ENTONCES
            Agregar obj.nombre a seleccionados
            pesoTotal ← pesoTotal + obj.peso
            valorTotal ← valorTotal + obj.valor
        FIN SI
    FIN PARA
    
    Mostrar "Mejor solución: Llevar", seleccionados
    Mostrar "Valor total: ", valorTotal, "(Peso total:", pesoTotal, "kg, sin superar los", capacidadMochila, "kg)."
FIN FUNCIÓN
