Ejemplo 8: Recorrido del Caballo en el Tablero

Inicio
    Definir tablero[8][8] como matriz de ajedrez
    Definir movimientos como lista de coordenadas

    // Posición inicial del caballo
    PosiciónX ← 7  // H6 → fila 7
    PosiciónY ← 5  // H6 → columna 5

    // Lista de movimientos del caballo
    movimientos ← [
        (6, 3), (7, 1), (5, 0), (3, 1), (1, 0), (0, 2), (1, 4), (0, 6),
        (2, 7), (4, 6), (6, 7), (5, 5), (7, 6), (6, 4), (7, 2), (6, 0),
        (4, 1), (2, 0), (0, 1), (1, 3), (0, 5), (1, 7), (3, 6), (5, 7),
        (6, 5), (7, 7), (5, 6), (4, 4), (5, 2), (7, 3), (6, 1), (4, 0),
        (3, 2), (1, 1), (0, 3), (1, 5), (0, 7), (2, 6), (4, 7), (6, 6),
        (7, 4), (6, 2), (7, 0), (5, 1), (3, 0), (2, 2), (3, 4), (5, 3),
        (4, 5), (2, 4), (4, 3), (3, 5), (2, 3), (0, 4), (1, 6), (3, 7),
        (3, 3)
    ]

    // Recorrer la lista de movimientos
    PARA CADA movimiento EN movimientos HACER
        PosiciónX ← movimiento.fila
        PosiciónY ← movimiento.columna
        Imprimir "Caballo se mueve a posición (", PosiciónX, ", ", PosiciónY, ")"
    FIN PARA
FIN
